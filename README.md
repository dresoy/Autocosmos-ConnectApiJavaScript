# Autocosmos Connectar a Api HTML-JavaScript
Ejemplo de como conectar a la Api de Autocosmos con JavaScript


## Instalción

```
- git clone https://github.com/dresoy/Autocosmos-ConnectApiJavaScript.git
- npm install
```

## Uso

> Simplemente abre Index.html

### Campos Claves

> * **Dominio:** El demonio a utilizar, debe tener en cuenta .ar, .mx...
> * **Application Key:** Otorgada por un gestor de cuentas.
> * **Application Secret:** Otorgada por un gestor de cuentas.
> * **Signature Cuenta:** Verificar en la administración de usuario.
> * **Endpoint:** Enpoint del API2 a utilizar.
> * **Metodo del request:** Verbos HTTP.
> * **Content del request:** JSON a ser enviado.

## Documentación Completa

La documentación completa de la API2 está disponible en https://www.autocosmos.com.ar/developers

Para el metodo de calculo de Auth, implementado aquí en JS, puede consultar https://www.autocosmos.com.ar/developers/help/hmac
